# Turkcell GYGY Kotlin Bankacılık Uygulaması

Bu proje, terminal (komut satırı) üzerinde çalışan, temel bankacılık fonksiyonlarını içeren basit ve etkili bir Kotlin uygulamasıdır.

## Özellikler
- **Güvenli Giriş:** Uygulama açılışta PIN kodu doğrulaması ister.
- **Bakiye Sorgulama:** Mevcut bakiyenizi anlık görün.
- **Para İşlemleri:** Para yatırma ve para çekme fonksiyonları.
- **Para Transferi:** Belirtilen bir IBAN adresine bakiye gönderim simülasyonu.
- **Döviz Çevirici:** TL bakiyesini USD veya EUR cinsinden hesaplayabilme.
- **Kredi Hesaplayıcı:** Alınan kredi tutarı ve vadeye göre toplam geri ödeme hesaplama.

## Nasıl Çalıştırılır?

1. Projeyi bilgisayarınıza indirin.
2. Terminal (PowerShell/CMD) açın.
3. Şu komutu kullanarak uygulamayı başlatın:
   ```bash
   java -jar main.jar
   ```

## Önemli Bilgiler
- **Default PIN:** `1234`
- **Başlangıç Bakiyesi:** `1000 TL`
